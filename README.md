#  vortextech-aiml-week1
# Vortex Tech AI/ML Internship – Week 1

## Data Cleaning and Exploratory Data Analysis (EDA)

This repository contains my Week 1 assignment for the **Vortex Tech AI & ML Internship Program 2026**. The objective of this task is to perform data cleaning and exploratory data analysis (EDA) on a real-world dataset using Python and the Pandas library.

---

## 📌 Objective

The goal of this project is to:

- Load and inspect a real-world dataset.
- Identify and handle missing values.
- Detect and remove duplicate records.
- Verify and correct data types where necessary.
- Generate summary statistics.
- Create visualizations to better understand the dataset.
- Document each step with clear explanations.

---

## 📂 Dataset

**Dataset Used:** Titanic Dataset

- **Source:** Kaggle
- **Rows:** 891
- **Format:** CSV

The Titanic dataset contains passenger information such as age, gender, passenger class, fare, and survival status.

## 📋 Tasks Completed

### ✅ Data Loading
- Imported the dataset using Pandas.
- Displayed the first few records.
- Inspected dataset structure and data types.

### ✅ Data Cleaning
- Checked for missing values.
- Filled missing values in numerical and categorical columns.
- Removed duplicate records.
- Verified data types.
- Removed unnecessary columns with excessive missing values.

### ✅ Exploratory Data Analysis
Generated summary statistics including:
- Mean
- Median
- Descriptive statistics
- Value counts

Created visualizations including:
- Histogram of passenger ages
- Survival count bar chart
- Gender distribution
- Fare distribution by passenger class
- Correlation heatmap

---

## 📊 Key Findings

- Most passengers were between **20 and 40 years old**.
- Male passengers outnumbered female passengers.
- The majority of passengers did not survive.
- First-class passengers generally paid higher fares.
- Missing values were successfully handled, resulting in a clean dataset suitable for further machine learning tasks.
